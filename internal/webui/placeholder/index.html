⚠ 前端尚未构建到二进制

当前 tshoot serve 已在运行，/api/* 接口可用，但前端静态资源还是占位。

方式一：构建前端并重新编译 Go

cd web && npm ci --ignore-scripts && npm run build
cp -R dist/. ../internal/webui/dist/
cd .. && go build -o bin/tshoot ./cmd/tshoot
./bin/tshoot serve

方式二：开发模式（两个进程）

保持 Go 后端运行，另起终端：

cd web && npm ci --ignore-scripts && npm run dev

Vite 会把 /api 请求代理到 Go 后端，热重载生效。
